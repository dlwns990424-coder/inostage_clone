*github depoly

1. 깃허브 저장소 세팅 이동

2. 옆에 메뉴중 pages 클릭

3. branch를 none -> main(또는 master)로 변경 후 저장

4. 저장소 페이지 이동하여 중간중간 새로고침 확인

5. 배포가 잘 진행중이면 오른쪽 중간에 Depolyments 메뉴가 생성됨

6. Depolyments에서 진행 상황을 볼 수 있고, 초록색 체크가 뜨면 배포 성공

7. 주소 클릭 하여 확인

-> 배포는 깃허브 업로드 된 저장소 기준으로 첫 배포시 세팅하는 시간이 있기 때문에 상황에 따라 배포속도는 다름

-> 파일이 수정되었으면 다시 한번 더 커밋 -> 푸시 진행하여 배포상태 확인

-> html 파일명은 반드시 index.html이어야 사이트 확인 가능

-> 배포 주소는 기본적으로 아이디명.github.io/저장소명으로 되어있음
